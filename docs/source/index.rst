Introduction
------------

**EasonDrone** 是一个自主四旋翼无人机框架，基于 **PX4-Autopilot**, **MAVROS**, **ROS** 等开源框架开发，集成了控制、定位、重建、规划、视觉等功能。

作为开源项目，欢迎各位使用、贡献。不完善之处，还请各位大佬多多批评指正！

.. note::

   This project is under active development.
